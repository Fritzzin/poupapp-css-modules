# Poupapp (CSS Modules)
Curso da Alura sobre CSS Modules em React. Me desafiei tentando utilizar Nextjs e Typescript para maior parte do projeto, porém o seu foco é na estilização.

## Getting Started

Para subir o servidor, basta instalar as dependências e rodar:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abrir [http://localhost:3000](http://localhost:3000) com seu browser.
